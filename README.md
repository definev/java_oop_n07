### Giao diện gửi - lấy xe

1. Tính năng gửi xe
   - Nhập biển số xe
   - Chọn khu vực gửi xe
   - Nhấn nút "Add"
   - GIF DEMO
   
![demo-gui-xe.gif](assets%2Fdemo-gui-xe.gif)

2. Tính năng lấy xe
   - Nhập biển số xe
   - Nhấn nút "Trả xe"
   - Nhấn nút "Trả" tính tiền trả xe
   - Khách hàng thanh toán
   - GIF DEMO

![demo-lay-xe.gif](assets%2Fdemo-lay-xe.gif)
